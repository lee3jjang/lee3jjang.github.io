## 스토리보드
* 스토리보드, 화면설계, 와이어프레임 비슷한 용어임
* 와이어프레임 : 선을 통해 완성된 모습을 연상할 수 있도록 하는 작업들
* 화면설계 : 레이아웃, 그래픽, 텍스트 요소 배치 크기 색상 표현 -> 화면 그림
* 스토리보드 : 와이어프레임 같이 선으로 그릴 수 있고, 화면표현으로도 가능함
* 스토리보드에는 정책, 규칙, 정보구조, 화면흐름, 사이트맵, 페이지 구조의 앱 서비스 기획 내용 담김

## 앱 서비스 기획 - 실전편
* (서비스정의) 스미스윌슨 보간결과를 볼 수 있는 앱
* (주요기능) 
    1. 데이터수집 (자동, backend)
    2. 시뮬레이션 기능
       * 모델링 : 만기, 변동성 조정(default = 0), LTFR, 수렴지점(default = max(llp+30, 60)?), 수렴속도(default = 1bp), 화폐(krw, usd), 날짜(불러오기 형태 -> 수기입력 가능), 현물/선도, 자산/부채
    3. 기타 기능
       * 설정, 결과 전송(이메일, 카톡), 로그인

* (정보마인드맵)
    1. 시뮬레이션 설정 화면 : 설정들 -> 실행버튼 -> 팝업창으로 완료 뜸
    2. 결과 메인 화면 : 결과목록 (리스트 형태) -> 하나 클릭하면 화면 나옴
       * 결과 화면 : 윗단에 그래프, 아랫단에 결과 명세, 사이에 결과(스크롤바 형태)
